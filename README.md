# PCA-Net
Code for PCA-Net: Progressive Co-Attention Network for Fine-Grained Visual Classification
[arXiv](https://arxiv.org/abs/2101.08527 "aeXiv")

## Requirement
python 3.6

PyTorch >= 1.1.1

torchvision >= 0.2.1

## Training
Train from scratch with `main.py`.
